
# Hangman Game

1. On running the ```main.py``` file, user with the main menu in the console.
2. The game gives you 7 life.
3. Take hint with ```0```. Taking hint will cause lose of 1 life.
4. You can change the some veriables like error message, instructions, number of lifes form the ```constants.py``` file.
5. User can play again using retry option.
